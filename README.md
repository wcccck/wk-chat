## 项目介绍
   这是一个vue3 + typescript + JSX的实战项目 帮助对vue的理解与学习 手写组件 未使用大型UI组件库  
   JSX语法可以在一个页面编写多个组件 vue3对typescript的支持度更高 可以更好的保持项目的健壮性
   项目使用postcss+amfe-flexible 做移动端适配
   SSE 实现用户聊天功能
   
## 项目技术栈
   vue3 + typescript + vite + jsx + axios + sass + flex + svg  
   node版本 v16.13.0  
   管理工具:yarn
